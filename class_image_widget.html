Member Enumeration Documentation

enum ImageWidget::ImageMode

This enum type defines whether the image should be automatically adjusted to the widget or not.

Enumerator:
FitToWidget 

The image will be automatically adjusted to the widget, keeping the acpect ratio in mind.

ManualAdjustment 

The image transformations like translation, rotation and scale will not be adjusted automatically.
